基本逻辑
1. 先设定一个区域，目标获取该区域的所有漫画书信息
2. 得到整个 list(68 item)，pagesize=68，pageindex=1~n
3. 把 68个书的信息存下来，然后逐个去看书的单页，取 tags 和 summary 两个信息
4. 取书的封面
5. 68 个书的信息取完后，再取下一个 list
